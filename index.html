Tentang Bootstrap!
CSS

CSS digunakan untuk mengatur tampilan dan gaya dari elemen-elemen HTML di halaman web.

Dengan CSS, dapat mengontrol warna, jenis huruf, margin, padding, dan banyak aspek tata letak dan desain halaman web.

HTML

HTML adalah bahasa markup yang digunakan untuk membuat struktur dasar dari halaman web.

HTML memberikan tag dan elemen untuk menentukan bagaimana konten web harus diorganisir, seperti teks, gambar, tautan, dan elemen-elemen lainnya.

JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi interaktif.

Dengan JavaScript, Anda dapat menangani peristiwa (events), memanipulasi DOM (Document Object Model), dan melakukan berbagai tugas di sisi klien (client-side) untuk meningkatkan pengalaman pengguna.

Contoh penggunaan JavaScript termasuk validasi formulir, animasi, dan interaksi dengan API.

Bootstrap V 5.3

Bootstrap adalah kerangka kerja (framework) front-end yang dikembangkan oleh Twitter.

Bootstrap menyediakan serangkaian alat dan komponen UI (User Interface) yang dapat digunakan untuk membangun halaman web yang responsif dan menarik.

Dengan Bootstrap, Anda dapat dengan cepat membuat tata letak dan desain yang responsif tanpa harus menulis banyak kode CSS dan JavaScript sendiri.

Contoh komponen Bootstrap termasuk grid system, navigasi, tombol, formulir, dan lainnya.
